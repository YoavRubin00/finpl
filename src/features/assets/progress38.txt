# PRD 38 — Real Assets Portfolio Progress

## Learnings (from prior PRDs)
- Zustand stores must use `persist` with `AsyncStorage` + `partialize` for selective persistence
- `useEconomyStore` exposes `addCoins(n)` and `spendCoins(n)` — `spendCoins` returns boolean (false if insufficient)
- ISO date key pattern for daily reset: `new Date().toISOString().slice(0, 10)` → "2026-03-11"
- Lottie files live in `assets/lottie/` — Money.json, Crown.json, gaming.json confirmed available
- `GlowCard` from `src/components/ui/GlowCard.tsx` accepts `glowColor` prop
- `AnimatedPressable` from `src/components/ui/AnimatedPressable.tsx` for spring-press buttons
- `DiamondBackground` wraps screens for consistent dark BG
- `useEntranceAnimation(fadeInUp, { delay })` pattern for staggered entrance animations
- Import pattern: `import { useEconomyStore } from "../economy/useEconomyStore"` (relative, no barrel index)
- No `any` — strict TypeScript; use `as const` for literal unions
- `CLASH` theme: `CLASH.goldBorder`, `CLASH.textShadow` etc. from `src/constants/theme.ts`
- `TITLE_TEXT`, `SUBTITLE_TEXT` from theme for consistent typography
