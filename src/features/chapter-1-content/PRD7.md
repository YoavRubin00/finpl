# PRD 7: תוכן האפליקציה בייצור - פרק 1: הישרדות (Survival) 🏕️

## 🌟 הקדמת הפרק: מה זה בכלל "הישרדות"?
ברוכים הבאים לג'ונגל הפיננסי. פרק 1 הוא הבייס של הכל – איך לא לטבוע כשכולם מסביב קונים באשראי וחיים במינוס. המטרה פה היא לא להפוך אתכם למיליונרים מחר בבוקר, אלא קודם כל ללמוד איך לשרוד את החודש, להבין טריקים שמנסים לעשות עליכם בסופר או בבנק, ולבנות "כריות אוויר" שיגנו על הכסף שלכם מול בלת"מים. בלי בילבולי שכל אקדמיים – רק תכלס אכזרי שיציל לכם את העו"ש. מוכנים לצאת מהמינוס? בואו נשחק.

## User Stories

### US-001: Content Data Types & Module Data
**Description:** As a developer, I want typed data structures for chapters, modules, flashcards, quizzes, and sim concepts so the lesson flow can render content dynamically.

**Acceptance Criteria:**
- [x] Create `types.ts` with interfaces: `Chapter` (id, title, description, pyramidLayer, modules[]), `Module` (id, title, videoHook, interactiveIntro, flashcards[], quizzes[], simConcept), `Flashcard` (id, text), `QuizQuestion` (id, question, options[], correctAnswer, successFeedback, failFeedback, type: 'multiple-choice' | 'scenario'), `SimConcept` (id, title, description)
- [x] Create `chapter1Data.ts` with all 9 modules (1-9) fully populated from PRD content below, typed as `Chapter`
- [x] Typecheck passes

### US-002: Chapter Progress Store (Zustand)
**Description:** As a developer, I want to track per-module completion, quiz scores, and current position in the lesson flow.

**Acceptance Criteria:**
- [x] Create `useChapterStore.ts` with state: `completedModules: string[]`, `quizResults: Record<string, { correct: number, total: number }>`, `currentModuleIndex: number`
- [x] Add actions: `completeModule(moduleId)`, `recordQuizAnswer(moduleId, questionId, isCorrect)`, `setCurrentModule(index)`
- [x] On module completion, grant XP + Coins via `useEconomyStore` (e.g., 30 XP + 10 Coins per module)
- [x] Persist state with Zustand persist middleware + AsyncStorage
- [x] Typecheck passes

### US-003: Lesson Flow Screen (Flashcards + Quizzes)
**Description:** As a user, I want to tap into a module and experience a flow of flashcards followed by quizzes, with feedback and animations.

**Acceptance Criteria:**
- [x] Create `LessonFlowScreen` component that receives a `moduleId` param
- [x] Display flashcards as swipeable/tappable cards (tap to advance), neon-styled with fade/slide animations
- [x] After flashcards, present quizzes one-by-one: show question + options as tappable buttons
- [x] On answer: green glow + success feedback text if correct, red shake + fail feedback if wrong
- [x] After all quizzes, show summary (X/Y correct) and grant XP/Coins, call `completeModule`
- [x] Styling matches FinPlay dark/neon theme, RTL-ready
- [ ] Typecheck passes
- [ ] Verify changes work in browser

### US-004: Chapter Map Screen (Module List)
**Description:** As a user, I want a chapter overview screen showing all 9 modules with my progress, so I can pick which module to study.

**Acceptance Criteria:**
- [ ] Create `ChapterMapScreen` component displaying chapter title, description, and a vertical list of module cards
- [ ] Each module card shows: number, title, completion status (locked/available/completed icon)
- [ ] Completed modules show a green checkmark, current module pulses with neon glow, future modules are dimmed/locked
- [ ] Tapping an available module navigates to `LessonFlowScreen` with that module's data
- [ ] Progress bar at top showing X/9 modules completed
- [ ] Typecheck passes
- [ ] Verify changes work in browser

### US-005: Navigation Integration
**Description:** As a user, I want to access Chapter 1 from the pyramid screen or a dedicated tab, so the content is discoverable.

**Acceptance Criteria:**
- [ ] Add route for `ChapterMapScreen` (e.g., accessible from PyramidScreen layer 1 tap or a "Start Learning" CTA)
- [ ] Add route for `LessonFlowScreen` with `moduleId` param
- [ ] Navigation flow: PyramidScreen → ChapterMapScreen → LessonFlowScreen → back to ChapterMap
- [ ] Typecheck passes
- [ ] Verify changes work in browser

## Content Data (Modules 1-9)

## Module 1: תזרים ותקציב (חוק 50/30/20)
🎬 Video Hook: מישהו בוכה מול היתרה באפליקציה של הבנק, בזמן ששליח וולט דופק בדלת עם מגש סושי ב-200 שקל.
🎯 Interactive Intro: מופיעה אנימציה של משפך עם 3 חורים – אחד ענק (50%), אחד בינוני (30%) ואחד קטן (20%). המשתמש מתבקש להכניס מטבע-משכורת חסר פוקוס למשפך ולראות איך הוא מתפצל אוטומטית למטרות. טקסט מלווה: "תזרים זה הכל. בואו נלמד איך לחלק את העוגה בלי להישאר רעבים ולהצליח גם לחסוך".
📇 Flashcard 1: חוק ה-50/30/20 הוא הצ'יט-קוד שלך לעו"ש. 50% מהמשכורת הולך לדברים שחייבים (שכירות, חשבונות, אוכל הביתה).
📇 Flashcard 2: 30% הולך לפאן (יציאות, בגדים, נטפליקס), וה-20% הנותרים? עפים ישר לחיסכון והשקעות כדי לקנות לך עתיד.
❓ Quiz 1 (Multiple Choice):

Question: נכנסה לך משכורת של 10K. כמה מותר לך לשרוף על "פאן ושטויות" לפי החוק?

Options: A) 5,000 ש"ח, B) 3,000 ש"ח, C) 2,000 ש"ח, D) הכל, חיים פעם אחת

Correct Answer: B

Success Feedback: 🔥 בול! 30% לפאן זה אחלה תקציב, תתפרע – אבל בגבולות הגזרה.

Fail Feedback: 💀 נסחפת. 30% זה המקסימום לפאן, אחרת תמצא את עצמך אוכל פריכיות בסוף החודש.
❓ Quiz 2 (Scenario-based):

Question: מאיפה משלמים על מנוי לחדר כושר שאתה הולך אליו (אולי) פעם בשבועיים?

Options: A) מה-50% של הצרכים, זה בריאות!, B) מה-30% של הפאן, C) מה-20% של החיסכון

Correct Answer: B

Success Feedback: 🎯 נכון. זה שזה ספורט לא הופך את זה להוצאת חובה קיומית כמו שכר דירה.

Fail Feedback: 🤡 הלוואי. חדר כושר, נטפליקס ומסעדות – הכל יושב על ה-30% של הפאן.
❓ Quiz 3 (Scenario-based):

Question: קיבלת בונוס פסיכי של 3,000 שקל בעבודה. חבר לוחץ לטוס לסופ"ש באתונה. מה עושים?

Options: A) טס וזורק את החוק מהחלון, B) מפצל לפי החוק: מפריש 600 לחיסכון וטס עם השאר, C) שם הכל בחיסכון ובוכה בבית

Correct Answer: B

Success Feedback: 👑 וואלה, גם נהנית וגם לא דפקת את העתיד שלך. זה ניהול כסף של בוס.

Fail Feedback: 🚩 כסף זה כסף. גם בונוסים נכנסים למשפך של החוק, אחרת תמיד תמצא תירוץ לבזבז.
❓ Quiz 4 (Multiple Choice):

Question: למה בכלל צריך להפריד בין "צרכים" (50%) ל"רצונות" (30%)?

Options: A) סתם כדי שיהיה טבלת אקסל יפה, B) כי אם לא תגדיר גבול, הרצונות שלך יאכלו את הכסף של שכר הדירה, C) כדי שהבנק יראה שאתה אחראי

Correct Answer: B

Success Feedback: 🧠 חשיבה חדה. בלי גבולות, כל ג'ינס הופך פתאום ל"צורך קיומי".

Fail Feedback: 🤦‍♂️ לא. אם לא תפריד ביניהם, אתה תגלה שהוצאת את כסף השכירות על כרטיסים להופעה.
🎮 SIM Concept: "טינדר הוצאות": נופלות הוצאות מהשמיים (המבורגר, חשבון חשמל, ביטוח) וצריך למיין אותן ל-3 סלים (50, 30, 20) לפני שהזמן נגמר.

## Module 2: מלכודת המינוס
🎬 Video Hook: מישהו מעביר כרטיס בסופר, שומע צפצוף סירוב וצועק "אבל יש לי מסגרת בחשבון!".
🎯 Interactive Intro: המשתמש רואה "מד-חיים" ירוק שיורד מתחת לאפס. אז צבעו הופך לאדום אזהרה ו"מפלצות ריבית חריגה" מתחילות לנגוס בו בהדרגה. טקסט מלווה: "ברוכים הבאים למינוס - המקום שבו אתה משלם כסף על כסף שאינך באמת הבעלים שלו".
📇 Flashcard 1: מינוס הוא לא "כסף של הבנק שמרשה לי להשתמש בו". זו פשוט הלוואה סופר-יקרה עם ריבית שקטה ששוחטת אותך כל חודש.
📇 Flashcard 2: "מסגרת אשראי" זה מושג שהבנק המציא כדי שתרגיש בנוח לבזבז כסף שאין לך. ירדת מתחת לאפס? התחלת לשלם.
❓ Quiz 1 (Multiple Choice):

Question: מה הבנק באמת מרגיש כשהוא רואה שאתה במינוס קבוע?

Options: A) הוא דואג לך ורוצה לעזור, B) הוא עף על זה כי אתה מייצר לו רווח נקי מריבית חריגה, C) הוא כועס עליך, D) לא אכפת לו

Correct Answer: B

Success Feedback: 💸 בוקר טוב! אתה הלקוח האהוב עליהם. מינוס = כסף קל לבנק.

Fail Feedback: 🚫 חמוד שאתה חושב ככה. הבנק מתפרנס מהריבית שאתה משלם על המינוס הזה.
❓ Quiz 2 (Scenario-based):

Question: אתה במינוס 5,000 ש"ח. הבנקאי מתקשר ומציע "הלוואה נוחה לסגירת המינוס ב-60 תשלומים". מסכים?

Options: A) ברור, זה סוגר לי את המינוס ואני נושם, B) מלכודת! זה רק לפנות מקום למינוס חדש ולשלם ריבית פעמיים, C) כן, כי הבנק מפסיד מזה

Correct Answer: B

Success Feedback: 🛡️ יפה שניטרלת את המוקש. לסגור חוב עם חוב זה הצעד הראשון לפשיטת רגל.

Fail Feedback: ☠️ נפלת בפח. סגרת את המינוס בהלוואה, ועוד חודשיים תהיה שוב במינוס בתוספת החזר הלוואה.
❓ Quiz 3 (Multiple Choice):

Question: מהי "ריבית חריגה"?

Options: A) הריבית הרגילה במשק, B) קנס מטורף שהבנק לוקח כשאתה חורג מהמסגרת שהוא אישר לך, C) ריבית שמקבלים על חיסכון טוב

Correct Answer: B

Success Feedback: 📉 עצוב אבל נכון. חרגת? שילמת ריבית שוק אפור בחסות החוק.

Fail Feedback: 🚩 הלוואי. חרגת מהמסגרת? הבנק קורע אותך בריבית שיכולה להגיע גם ל-15% ויותר.
❓ Quiz 4 (Scenario-based):

Question: יש לך יתרה של 100 שקל. חברים מזמינים למסעדה שתעלה 150. "נחזיר בחודש הבא", אתה אומר לעצמך. ללכת?

Options: A) ברור, כולה 50 שקל מינוס, B) לא, ככה מתחיל כדור שלג. אם אין כסף בעו"ש, אין מסעדה.

Correct Answer: B

Success Feedback: 🛑 משמעת ברזל. להגיד "לא" היום שווה הרבה כסף מחר.

Fail Feedback: 🍔 ככה מתחילות צרות. ה-50 שקל האלה יהפכו ל-500 בחודש הבא.
🎮 SIM Concept: משחק הרצה (Endless Runner): אתה עובר בתוך מנהרה (העו"ש). ירידה מתחת לקו האפס גורמת ל"מפלצות ריבית" לנגוס בך ולהאט אותך.

## Module 3: אשראי ותשלום מינימום
🎬 Video Hook: בחורה משלמת על אייפון חדש ב-36 תשלומים ואומרת למצלמה: "בקטנה, זה רק 150 שקל בחודש!".
🎯 Interactive Intro: מופיע כפתור "שלם עכשיו מינימום!". ברגע שהמשתמש לוחץ עליו, "כדור שלג" רותח גדל מאחורי המסך ומאיים למחוץ את היתרה. טקסט מלווה: "תשלום מינימום או קרדיט? זאת פשוט הלוואה מוסווית שמתנפחת במחשכים עמוק בתוך כרטיס האשראי שלך".
📇 Flashcard 1: "קרדיט" ו"תשלום מינימום" הן המילים הכי מסוכנות בעולם הפיננסי. זו דרך לגרום לך לשלם על ג'ינס ב-200 שקל קרוב ל-300 שקל.
📇 Flashcard 2: לשלם את המינימום בכרטיס האשראי זה כמו לשים פלסטר על עורק מדמם. החוב המקורי נשאר והריבית מתנפחת מיום ליום.
❓ Quiz 1 (Multiple Choice):

Question: מה ההבדל בין פריסה ל"תשלומים רגילים" לבין פריסה ב"קרדיט"?

Options: A) אין הבדל, זה רק שם אחר, B) תשלומים רגילים זה ללא ריבית (על חשבון העסק), בקרדיט אתה משלם ריבית גבוהה לחברת האשראי, C) קרדיט זה רק לאנשי עסקים

Correct Answer: B

Success Feedback: 💳 אלוף הפלסטיק. פריסה בקרדיט היא פשוט הלוואה יקרה בתחפושת.

Fail Feedback: 🤦‍♀️ טעות קשה. קרדיט = הלוואה בריבית גבוהה. תמיד תעדיף תשלומים רגילים אם כבר חילקת.
❓ Quiz 2 (Multiple Choice):

Question: מה המשמעות האמיתית של ללחוץ על כפתור "תשלום מינימום" באפליקציית האשראי?

Options: A) עקצת את המערכת, B) אתה מורח את החוב למחר, ומשלם עליו 10%-15% ריבית, C) זה מקפיא לך את החשבון

Correct Answer: B

Success Feedback: 🎯 בול. המינימום עכשיו זה המקסימום נזק לעתיד שלך.

Fail Feedback: 🚩 אזהרת מסע! תשלום מינימום סוחב את החוב שלך קדימה עם ריבית רצחנית.
❓ Quiz 3 (Scenario-based):

Question: עשית קנייה ענקית לדירה. חברת האשראי מסמסת: "כבדה עליך ההוצאה? החזר רק 500 ש"ח החודש והשאר יידחה". מסכים?

Options: A) איזה שירות לקוחות מדהים, ברור שכן, B) מתעלם מההודעה. החיוב ירד במלואו ואספוג את זה החודש במקום לשלם ריבית.

Correct Answer: B

Success Feedback: 🛡️ התחמקת מכדור! ה"טובה" הזו שלהם עולה הרבה מאוד כסף.

Fail Feedback: 💸 אל תהיה תמים, הם לא חברים שלך. ה"דחייה" הזו היא הלוואה לכל דבר ועניין.
❓ Quiz 4 (Scenario-based):

Question: אתה רוצה לקנות נעליים ב-600 שקל. באיזה תרחיש תרגיש פחות כאב פסיכולוגי בזמן הקנייה?

Options: A) כשתשלם בשטרות של 200, B) כשתעביר אפל-פיי בשעון בלי להסתכל, C) אותו דבר

Correct Answer: B

Success Feedback: ⌚ מחקרים מוכיחים: ככל שהתשלום יותר "וירטואלי", אנחנו מרגישים פחות כאב ומוציאים יותר.

Fail Feedback: 💳 לא נכון. כסף מזומן כואב יותר. לשלם באפל-פיי מרגיש כמו לשחק במונופול ולכן מבזבזים יותר.
🎮 SIM Concept: כדור שלג דיגיטלי: אתה פורס תשלומים על מסך, ורואה בזמן אמת איך כדור השלג של הריבית גדל ומוחץ את המשכורת העתידית שלך.

## Module 4: ריבית דריבית (הקסם השמיני)
🎬 Video Hook: בחור מציג גרף שנראה כמו קו ישר, פתאום יש דרופ בשיר והגרף ממריא בזווית של 90 מעלות לחלל.
🎯 Interactive Intro: שני עציצים וירטואליים. המשתמש מחליט "להשקות" פעם אחת את העץ ואז מפעיל כפתור Fast-Forward ורואה אותו מטפס באופן אקספוננציאלי. העץ שלא הושקה נשאר דקיק. טקסט מלווה: "ריבית דריבית היא הקסם השקט שבו הכסף מתחיל לעבוד בשבילך, בזמן שאתה ישן".
📇 Flashcard 1: אלברט איינשטיין קרא לה "הפלא השמיני של תבל". ריבית דריבית זה כשהכסף שלך עושה רווח, ואז הרווח מתחיל לעשות רווח בעצמו.
📇 Flashcard 2: המרכיב הסודי כאן הוא לא כמה כסף יש לך, אלא כמה זמן אתה נותן לו לעבוד. תתחיל מוקדם, תעבוד פחות קשה אחר כך.
❓ Quiz 1 (Multiple Choice):

Question: מה הפקטור הכי קריטי כדי שריבית דריבית באמת תעיף לך את התיק לחלל?

Options: A) להתחיל להשקיע כמה שיותר צעיר, B) לשים מליון שקל מכה ראשונה, C) לבחור את המניה הכי מסוכנת

Correct Answer: A

Success Feedback: 🚀 בארור! "זמן בשוק" מנצח "תזמון של השוק".

Fail Feedback: 📉 לאט לך. גם עם 100 שקל בחודש, ברגע שיש מספיק זמן, המנוע הזה משתולל. התשובה היא זמן.
❓ Quiz 2 (Scenario-based):

Question: נטע (בת 22) משקיעה 500 ש"ח בחודש. אורי אומר: "אני אתחיל בגיל 35, כשתהיה לי משכורת קטלנית ואשים אלפייה". למי יהיה יותר בגיל 60?

Options: A) לאורי כי הוא מפקיד פי 2 בכל חודש, B) לנטע, כי לכסף שלה היו עוד 13 שנה לצבור ריבית על ריבית

Correct Answer: B

Success Feedback: 💸 נטע היא המלכה. הזמן שעבר נותן לה יתרון פסיכי שאורי יתקשה מאוד להדביק.

## Module 5: לקרוא תלוש שכר (ברוטו vs נטו)

## Module 8: מלכודות שיווקיות בסופר
🎬 Video Hook: בחור נכנס לסופר לקנות רק חלב, ויוצא עם עגלה מלאה כולל מכשיר להכנת פונדו במבצע.

📇 Flashcard 2: "מבצע" של 1+1 לא תמיד חוסך לך כסף. אם באת לקנות חטיף ב-10 וקנית שניים ב-15 – לא חסכת 5 שקלים, הוצאת 5 שקלים יותר ממה שתכננת.

Fail Feedback: 😵 מהנדסי התנהגות עובדים על זה חודשים. אתה קונה עגבניות, מרגיש בריא, ואז מרשה לעצמך שוקולד בלי רגשות אשם.

Options: A) מסתכלים איזה אריזה נראית גדולה יותר, B) מסתכלים על המדבקה הקטנה של "מחיר ל-100 גרם", C) לפי מה שבמבצע

Success Feedback: ⚖️ נשק סודי. המחיר ל-100 גרם מנפץ את כל אשליות האריזה של היצרנים.

Fail Feedback: 🚫 לא. חברות מקטינות אריזות (שרינקפלציה). רק המחיר ל-100 גרם מראה את האמת.
❓ Quiz 4 (Scenario-based):

Question: פופקורן קטן בקולנוע עולה 25 ש"ח, ענק עולה 35 ש"ח. למה הבינוני עולה 33 ש"ח?

Options: A) כי זה המחיר האמיתי שלו, B) זו טעות של המוכר, C) "אפקט הפיתיון" – הבינוני שם רק כדי שתגיד "עדיף כבר להוסיף 2 שקלים לענק"

Correct Answer: C

Success Feedback: 🍿 יואו, הבנת את המטריקס. הם פשוט דחפו אותך לקנות את היקר ביותר.

Fail Feedback: 🎭 נפלת באפקט הפיתיון! הבינוני נועד להיראות לא משתלם כדי שתקנה את הענק בהרבה יותר כסף.
🎮 SIM Concept: מירוץ עגלות קניות: דוחפים עגלה וצריך לאסוף רק מוצרי בסיס ולחמוק משלטי ניאון ענקיים של "מבצע!" שמאיטים אותך.

## Module 9: קרן חירום (הכרית שלך)
🎬 Video Hook: הרכב מתקלקל, המסך של הטלפון נשבר והדוד מתפוצץ... הכל באותו שבוע. מאיפה לוקחים כסף?!
🎯 Interactive Intro: דמות נופלת באוויר בזמן סדרה של "באמפרים מחיי היום-יום" (פיקשוש שיניים, תיקונים בדוד). המשתמש לוחץ מהר להפעיל "כרית אוויר כחולה" (קרן חירום) שתופסת אותה ברכות שנייה לפני פגיעה במשטח אדום של "הלוואות בתנאים נוראיים". טקסט מלווה: "קרן חירום היא האפוד המגן שאתה שומר למקרה צרה, מחוץ להישג ידו של הדחף לבזבז. חייבים רשת ביטחון".
📇 Flashcard 1: החיים קורים ותקלות עולות ים כסף מזומן. קרן חירום היא פק"מ בצד שיציל אותך מלקיחת הלוואות חונקות כשהמקרר חוטף התקף לב.
📇 Flashcard 2: כמה צריך שם? בערך 3 חודשי הוצאות. הכלל הכי חשוב: לא נוגעים בכסף הזה כדי לטוס ליוון. זה נטו לקטסטרופות.
❓ Quiz 1 (Multiple Choice):

Question: מה הגודל האידיאלי של קרן חירום לצעיר ממוצע?

Options: A) סכום שווה ל-3 עד 6 חודשי הוצאות מחיה בסיסיות, B) מספיק בשביל לפתוח שולחן מחר, C) מאה אלף שקל

Correct Answer: A

Success Feedback: 🚒 אשכרה. מרווח נשימה אמיתי ליום שבו מפטרים אותך או שיש בלת"ם.

Fail Feedback: 💥 לא. הקרן צריכה להחזיק אותך מעל המים כמה חודשים אם פתאום אין משכורת. 3 עד 6 חודשי הוצאות.
❓ Quiz 2 (Scenario-based):

Question: חסכת 15K בקרן חירום. פתאום מצאת דיל של החיים למלדיביים ב-4K. מותר להשתמש בה?

Options: A) ברור, זה "חירום למצב הרוח", B) לא נוגעים! הקרן היא לטיפולי שיניים או פיטורים, לא לריזורטים.

Correct Answer: B

Success Feedback: 🛡️ אחראי רצח. קרן חירום היא כרית ביטחון, טיסה זה מותרות. (חפש תקציב ב-30% של הפאן).

Fail Feedback: 🚫 חטפת כדור! ברגע שתשרוף את זה על טיסה, המזגן יתקלקל ותיקח הלוואה.
❓ Quiz 3 (Multiple Choice):

Question: איפה הכי חכם להחזיק את קרן החירום שלך?

Options: A) במניות קריפטו מסוכנות כדי שתצמח, B) בפיקדון בנקאי רגיל או קרן כספית – היכן שהכסף נזיל וזמין ללא סיכון

Correct Answer: B

Success Feedback: 💧 נזילות היא שם המשחק פה. הכסף צריך להיות זמין תוך רגע, לא נעול או בסיכון.

Fail Feedback: 📉 טעות אנושה. כסף של חירום לא מסכנים בבורסה, כי אולי כשתצטרך אותו השוק יהיה בירידה.
❓ Quiz 4 (Multiple Choice):

Question: איזה מהמקרים הבאים מוגדר כ"מצב חירום" שמצדיק שבירה של הקרן?

Options: A) אייפון 16 יצא וממש בא לך להשתדרג, B) הלפטופ שלך שאתה עובד איתו נשפך עליו קפה ומת, C) יש סייל מטורף בזארה

Correct Answer: B

Success Feedback: 💻 בדיוק. כלי עבודה שנהרס פתאום או עניין רפואי הם ההגדרה המילונית לחירום.

Fail Feedback: 🤦‍♀️ באמת? קניות ומותגים זה לא חירום, זה פינוק. שמור את הקרן למשברים אמיתיים.
🎮 SIM Concept: משחק "טרמפולינה": נופלות "הוצאות אסון" (פנצ'ר, חשבון שיניים) ואתה מותח מתחתן רשת ביטחון (הקרן) שמקפיצה אותן החוצה ושומרת על מד העו"ש שלך.
