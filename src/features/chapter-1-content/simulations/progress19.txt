## Iteration 3 — US-003: Build simulation logic hook
- Created `useCompoundSim.ts` with compound interest calculation (monthly compounding)
- Files affected: `simulations/useCompoundSim.ts`
- Formula: FV = P(1+r/n)^(nt) + PMT×[((1+r/n)^(nt)-1)/(r/n)]
- Hook exposes: state, updateInitialAmount, updateMonthlyContribution, updateYears, reset
- Learnings for next cycles:
  - This sim is NOT round-based like Budget/MinusTrap/Snowball — it's an interactive visualizer
  - No score/grade system in the hook itself (summary handled in US-006)
  - Used useMemo for derived calculations, useCallback for handlers
  - Clamped inputs: years to [minYears, maxYears], amounts to >= 0, all rounded to integers
  - Pre-existing TS errors only in ArenaRankBadge — same as before
