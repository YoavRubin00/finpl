- Chapter data follows pattern: types.ts re-exports from chapter-1-content/types.ts, data file imports Chapter from local types.ts

- Flashcard IDs: fc-{chapter}-{module}-{seq}, Quiz IDs: q-{chapter}-{module}-{seq}, Sim IDs: sim-{chapter}-{module}
- correctAnswer is 0-indexed (A=0, B=1, C=2, D=3)
- Quiz type must be 'multiple-choice' | 'scenario' matching the PRD labels
