כמוב# Progress Log — PRD 47: Simulator Visual Overhaul

## Learnings

### effectiveGlowColor pattern (GlowCard)
When adding override props to existing components, compute a resolved value at the top of the function body (e.g. `const effectiveGlowColor = chapterGlow ?? glowColor`) and use that resolved value inside hooks. This avoids using conditional logic inside `useAnimatedStyle` which would violate React hooks rules.

### SimLottieBackground gradient type
`LinearGradient` from `expo-linear-gradient` accepts `readonly string[]` (or `[string, string, ...string[]]`). When combining a 2-element `chapterColors` tuple into a gradient, simply spread both colors — no need to repeat for a 3-stop gradient.

### getChapterTheme fallback
Since `OCEAN_CHAPTER_PALETTE` is `as const`, casting it to `Record<string, ChapterTheme>` is the cleanest way to allow dynamic key lookup with a `??` fallback, without TypeScript widening the return type.
