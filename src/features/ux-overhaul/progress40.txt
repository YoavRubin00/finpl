# Progress Log — PRD 40: Duo-Style Migration & Adaptive Theme

## Learnings
(Notes discovered during implementation)
- useTheme() hook at src/hooks/useTheme.ts returns AppTheme interface
- Light mode uses DUO from src/constants/theme.ts; Dark mode uses local DARK const in useTheme.ts
- AppBackground already uses useTheme() — good reference for pattern
- Pattern: `const theme = useTheme();` then `{ backgroundColor: theme.bg, color: theme.text }`
- GlobalWealthHeader has hard-coded: #ffffff (bg), #e5e7eb (border), #1f2937 (text)
- AnimatedTabBar has hard-coded: #fafafa (bg), static per-tab colors
- SupercellButton DUO_COLORS is light-only, needs dark variant
- Quiz/Lesson screens must NOT be changed (Clash style preserved)
- DUO palette keys: bg, surface, border, text, textMuted, green, greenDark, greenSurface, blue, blueDark, blueSurface, orange, orangeDark, orangeSurface, red, redDark, purple, purpleDark
